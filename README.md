# have-a-break-api
Created Api's for Have a break website using the php-codeigniter framework

Use the database provided to import and study - https://www.haveabreak.co.in/

Created API's are as follows:
'send-otp'
'verify-otp'
'create-agent'
'add-agent-details'
'get-agent-details'
'delete-agent'
'kyc-doc'
'upload-profilepic'
'reset-password'
'get-kyc-status'
'get-agent-packages'
'view-package-detials'
'create-lead'
'add-lead-followup'
'my-bookings'
